## Usage
![Insomnia](https://img.shields.io/badge/Insomnia-5849be?style=for-the-badge&logo=Insomnia&logoColor=white)

Use `insomnia` or `postman` execute the api calls to achieve the following:

USERS:
- Find All Users
- Find User by ID
- Add User
- Update User
- Delete User

FRIENDS:
- Add Friend
- Remove Friend

REACTIONS:
- Add Reaction
- Remove Reaction

THOUGHTS:
- Find All Thoughts
- Find Thought by ID
- Add Thought
- Update Thought
- Remove Thought
